Core Capabilities

Built for SREs who need
answers, not more dashboards.

🔁

Actor / Critic Agent Loop

The Actor proposes root-cause hypotheses; the Critic scores and rejects them with confidence thresholds. Rejected hypotheses feed back into prompt template fine-tuning — the engine gets sharper over time.

⚡

Streaming RCA Pipeline

Incident signal flows from Prometheus/Cloud Ops → Pub/Sub → Eventarc → Reflexion Engine. Embedding search, hypothesis generation, and ranked output all under 10 seconds P99. Zero polling.

🛡️

Human-in-the-Loop Gate

Any Terraform or kubectl action affecting more than 5 nodes triggers a Cloud Workflows approval state machine. Durable, auditable, integrated with Slack — not a webhook hack.

💸

Token Cliff Guard

A dynamic context window manager prunes conversation history and selectively re-inserts relevant prior turns. Pair with Cloud Billing API budget alerts — prevents the $50k → $220/mo horror story.

🔀

Gemini + Claude Routing

A model router scores query complexity and sends broad sweeps to Gemini 2.0 Flash ($0.075/M tokens) while escalating deep RCA to Claude Sonnet. 40–60% cost reduction with no quality loss.

📋

Reflexion Replay

Post-incident timeline showing every hypothesis the engine considered — and why each was accepted or rejected. Your compliance team will love it. Your future self will too.
